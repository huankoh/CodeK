# Manuscript figure materials

This directory contains the notebooks, aggregate source tables, and rendered outputs used for manuscript figures. Jupyter checkpoint copies and operating-system metadata are excluded from the archival release.

Run notebooks from the repository root so that relative paths resolve consistently. Some notebooks require locally obtained dataset inputs described in [`../DATA_AVAILABILITY.md`](../DATA_AVAILABILITY.md). Do not embed restricted row-level data in notebook outputs before publishing a new release.
